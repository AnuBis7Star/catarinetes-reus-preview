# Catarinetes Reus Preview

Static preview website in Catalan for a local women’s clothing shop in Reus, presenting looks, shop information, shipping/returns and contact details.

## Live demo

Add the GitHub Pages, Netlify or Vercel link here after deployment.

Example:

```text
https://your-github-username.github.io/catarinetes_reus_preview/
```

## Project type

- Business type: Women’s fashion shop
- Status: Preview / concept
- Language/content: mainly Catalan
- Structure: static website

## Features

- Fashion-focused landing page
- Look/gallery cards
- Shop information and opening hours
- Shipping and returns sections
- Contact and map area
- Responsive static layout

## Technologies

- HTML5
- CSS3
- Vanilla JavaScript

## How to run locally

Open `index.html` directly in your browser, or run a small local server from this folder:

```bash
python3 -m http.server 5500
```

Then open:

```text
http://localhost:5500
```

## How to publish with GitHub Pages

1. Create a new GitHub repository named `catarinetes_reus_preview`.
2. Upload all files from this folder to the repository.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch **main** and folder **/root**.
6. Save and wait until GitHub gives you the public URL.

## Notes before using with a real client

- Texts, images and policy details should be confirmed with the business before publishing.
- Useful as a client demo for local fashion/retail stores.
